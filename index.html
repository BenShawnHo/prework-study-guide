HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element separates the page into topical sections.
  • The ul element creates an unordered list using bullets.
  • The li element creates an item in the list using the text between the tags.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the latest main branch from GitHub
  • git add -A: adds changes to the staging area
  • git commit -m "message": commits changes to the version history
  • git push origin branch-name: pushes changes to the remote branch
